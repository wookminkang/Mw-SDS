# Select

목록에서 하나의 값을 선택할 때 사용하는 컴포넌트입니다.

---

## 1. 사용법 (Usage)

```tsx
import { Select } from "@/components/ui/select";

function Example() {
  return (
    <Select label="직업 선택">
      <option value="developer">개발자</option>
      <option value="designer">디자이너</option>
      <option value="planner">기획자</option>
    </Select>
  );
}
```

## 2. 상태 (States)

입력창과 동일하게 다양한 상태를 지원합니다.

<div className="flex flex-col gap-6 p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Select label="기본 상태">
    <option value="">선택해주세요</option>
    <option value="1">옵션 1</option>
    <option value="2">옵션 2</option>
  </Select>
  
  <Select label="성공 상태" success="정상적으로 선택되었습니다.">
    <option value="1">선택된 옵션</option>
  </Select>
  
  <Select label="에러 상태" error="필수 선택 항목입니다.">
    <option value="">선택해주세요</option>
    <option value="1">옵션 1</option>
  </Select>
  
  <Select label="비활성화 상태" disabled>
    <option value="1">선택 불가</option>
  </Select>
</div>

```tsx
<Select label="Default">...</Select>
<Select label="Success" success="Message">...</Select>
<Select label="Error" error="Message">...</Select>
<Select label="Disabled" disabled>...</Select>
```

---

## 3. 크기 (Sizes)

`sm`, `md`, `lg` 세 가지 크기를 지원합니다.

<div className="flex flex-col gap-6 p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Select size="sm">
    <option>Small Size</option>
  </Select>
  <Select size="md">
    <option>Medium Size (Default)</option>
  </Select>
  <Select size="lg">
    <option>Large Size</option>
  </Select>
</div>

```tsx
<Select size="sm">...</Select>
<Select size="md">...</Select>
<Select size="lg">...</Select>
```

---

## Props

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `label` | `string` | `-` | 셀렉트박스 상단에 표시될 라벨입니다. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 셀렉트박스의 크기를 설정합니다. |
| `error` | `boolean \| string` | `false` | 에러 상태를 표시하거나 에러 메시지를 출력합니다. |
| `success` | `boolean \| string` | `false` | 성공 상태를 표시하거나 성공 메시지를 출력합니다. |
| `helpText` | `string` | `-` | 셀렉트박스 하단에 표시될 도움말 텍스트입니다. |
| `disabled` | `boolean` | `false` | 셀렉트박스를 비활성화합니다. |
