# TableRow

정보의 제목(Key)과 내용(Value)을 나란히 배치하여 데이터를 간결하고 읽기 쉽게 표시할 때 사용합니다. 주로 상세 페이지의 정보 요약이나 설정 화면에서 유용합니다.

---

## 1. 사용법 (Usage)

```tsx
import { TableRow } from "@/components/ui/table-row";

function Example() {
  return (
    <TableRow 
      left="사용자 이름" 
      right="홍길동" 
    />
  );
}
```

## 2. 정렬 (Alignment)

내용의 높이가 다를 경우 `align` 속성을 통해 수직 정렬을 조절할 수 있습니다.

<div className="p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6 space-y-0">
  <TableRow align="space-between" left="스마트스코어" right="받는 분" />
  <TableRow align="space-between" left="스마트스코어" right="결제할 분" />
  <TableRow align="space-between" left="골프왕" right="동반자" />
</div>

```tsx
  <TableRow align="space-between" left="스마트스코어" right="받는 분" />
  <TableRow align="space-between" left="스마트스코어" right="결제할 분" />
  <TableRow align="space-between" left="골프왕" right="동반자" />
```

---

## 3. 비율 조절 (Left Ratio)

`leftRatio` 속성을 통해 왼쪽 레이블 영역의 너비 비율(%)을 자유롭게 조정할 수 있습니다.

<div className="p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <TableRow align="left" left="강스스" right="내용 영역이 더 넓게 표시됩니다." leftRatio={50}  />
  <TableRow align="left" left="욱스스" right="좌우가 정확히 절반씩 차지합니다." leftRatio={10 } />
</div>

```tsx
<TableRow left="Label (20%)" right="..." leftRatio={20} />
<TableRow left="Label (50%)" right="..." leftRatio={50} />
```

## 4. 다양한 활용 (Examples)

컴포넌트를 중첩하여 풍부한 정보를 표현할 수 있습니다.

<div className="p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <TableRow 
    left="현재 상태" 
    right={<Badge variant="success" isFilled isPill>활성화됨</Badge>} 
  />
  <TableRow 
    left="최근 접속" 
    right="2024-01-06 14:30:22" 
  />
  <TableRow 
    left="설명" 
    right="디자인 시스템 컴포넌트를 사용하여 구축된 대시보드 인터페이스 예시입니다." 
  />
</div>

```tsx
<TableRow 
  left="현재 상태" 
  right={<Badge variant="success" isFilled isPill>활성화됨</Badge>} 
/>
```

## Props

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `left` | `ReactNode` | `-` | 왼쪽에 표시될 레이블/제목입니다. |
| `right` | `ReactNode` | `-` | 오른쪽에 표시될 내용/데이터입니다. |
| `align` | `'top' \| 'center' \| 'bottom'` | `'center'` | 수직 정렬 방식을 설정합니다. |
| `leftRatio` | `number` | `30` | 왼쪽 레이블 영역의 너비 비율(%)을 설정합니다. |
