# Tabs

한 화면 내에서 콘텐츠를 그룹화하고 선택적으로 보여줄 때 사용하는 컴포넌트입니다.

---

## 1. 사용법 (Usage)

```tsx
import { Tabs } from "@/components/ui/tabs";

const items = [
  { id: 'tab1', label: '기본 정보', content: <div>기본 정보 콘텐츠입니다.</div> },
  { id: 'tab2', label: '상세 내역', content: <div>상세 내역 콘텐츠입니다.</div> },
];

function Example() {
  return <Tabs items={items} />;
}
```

## 2. 변체 (Variants)

### Line (Default)
하단에 인디케이터 라인이 있는 깔끔한 스타일입니다.

<div className="p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Tabs 
    items={[
      { id: '1', label: '전체', content: <p className="mt-2 text-slate-600">전체 목록을 보여줍니다.</p> },
      { id: '2', label: '입금', content: <p className="mt-2 text-slate-600">입금 내역만 보여줍니다.</p> },
      { id: '3', label: '출금', content: <p className="mt-2 text-slate-600">출금 내역만 보여줍니다.</p> },
    ]} 
  />
</div>

```tsx
<Tabs variant="line" items={items} />
```

### Pill
배경색이 있는 알약 모양의 스타일로, 옵션 선택 등에 유용합니다.

<div className="p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Tabs 
    variant="pill"
    items={[
      { id: '1', label: '오늘' },
      { id: '2', label: '1주일' },
      { id: '3', label: '1개월' },
      { id: '4', label: '3개월' },
    ]} 
  />
</div>

```tsx
<Tabs variant="pill" items={items} />
```

---

## 3. 제어 컴포넌트 (Controlled)

`activeTab`과 `onChange` 프롭을 통해 외부 상태와 연동할 수 있습니다.

```tsx
const [active, setActive] = useState('tab1');

<Tabs 
  activeTab={active} 
  onChange={setActive} 
  items={items} 
/>
```

---

## Props

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `items` | `TabItem[]` | `-` | 탭 목록 정보 (id, label, content) 입니다. (필수) |
| `activeTab` | `string` | `-` | 현재 활성화된 탭의 id입니다. |
| `onChange` | `(id: string) => void` | `-` | 탭이 변경될 때 호출되는 콜백 함수입니다. |
| `variant` | `'line' \| 'pill'` | `'line'` | 탭의 스타일 변체를 설정합니다. |
