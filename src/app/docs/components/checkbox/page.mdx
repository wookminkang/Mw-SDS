# Checkbox

사용자가 하나 이상의 옵션을 선택할 수 있도록 하는 입력 컴포넌트입니다.

---

## 1. 사용법 (Usage)

```tsx
import { Checkbox } from "@/components/ui/checkbox";

function Example() {
  return <Checkbox label="동의합니다" />;
}
```

## 2. 타입 (Types)

`Circle` 형태와 `Line` 형태 두 가지 스타일을 지원합니다.

### Circle (Default)
가장 일반적인 형태의 체크박스로, 선택 시 배경색이 채워집니다.

<div className="flex flex-wrap gap-8 p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Checkbox variant="circle" label="미선택" />
  <Checkbox variant="circle" label="선택됨" defaultChecked />
  <Checkbox variant="circle" label="비활성화" disabled />
</div>

```tsx
<Checkbox variant="circle" label="Circle Checkbox" />
```

### Line
테두리 없이 체크 아이콘의 색상만 변경되는 간결한 형태입니다.

<div className="flex flex-wrap gap-8 p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Checkbox variant="line" label="미선택" />
  <Checkbox variant="line" label="선택됨" defaultChecked />
  <Checkbox variant="line" label="비활성화" disabled />
</div>

```tsx
<Checkbox variant="line" label="Line Checkbox" />
```

---

## 3. 상태 (States)

<div className="flex flex-col gap-4 p-6 bg-white dark:bg-slate-900 rounded-xl border border-slate-200 dark:border-slate-800 my-6">
  <Checkbox label="기본 상태" />
  <Checkbox label="비활성화 상태" disabled />
  <Checkbox label="비활성화 + 선택 상태" disabled defaultChecked />
</div>

---

## Props

| 속성 | 타입 | 기본값 | 설명 |
| :--- | :--- | :--- | :--- |
| `variant` | `'circle' \| 'line'` | `'circle'` | 체크박스의 스타일 타입을 설정합니다. |
| `label` | `ReactNode` | `-` | 체크박스 옆에 표시될 라벨입니다. |
| `checked` | `boolean` | `-` | 체크 여부를 제어합니다. |
| `disabled` | `boolean` | `false` | 체크박스를 비활성화합니다. |
