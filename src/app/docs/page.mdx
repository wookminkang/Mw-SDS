# 문서 시작하기

이곳은 당신의 문서 홈 페이지입니다.

## 시작하는 방법
1. `src/app/docs` 폴더 안에 `.mdx` 파일을 만듭니다.
2. `src/components/Sidebar.tsx`에서 링크를 추가합니다.
3. 멋진 문서를 작성합니다!

## 레이아웃 안내1
현재 이 레이아웃은 다음 기능을 포함하고 있습니다:
- **Sticky Sidebar**: 왼쪽 사이드바는 항상 고정되어 있습니다.
- **Responsive Design**: 모바일에서는 사이드바가 숨겨지거나 조절되도록 설정할 수 있습니다.
- **Typography 스타일링**: `prose` 클래스를 통해 제목, 본문, 코드 블록 등이 자동으로 스타일링됩니다.
