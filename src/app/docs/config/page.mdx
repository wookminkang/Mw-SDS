# 기본 설정

MWDS의 스타일과 변수가 프로젝트에 정상적으로 적용되도록 테마 설정을 추가해야 합니다.

---

## 1. 테마 설치 및 설정

MWDS는 전역 CSS 변수를 통해 색상과 스타일을 관리합니다. 모든 컴포넌트가 정상적으로 보이게 하려면 `@mwds/theme`를 설치하고 최상위 파일에서 임포트해야 합니다.

### 설치

```bash
npm install @mwds/theme
```

### 글로벌 스타일 임포트

프로젝트의 최상위 CSS 파일(예: `src/app/globals.css` 또는 `src/main.tsx`)의 **가장 상단**에 테마 스타일을 불러옵니다.

```css
/* globals.css */
@import "@mwds/theme/src/index.css";

/* 여기에 사용자 정의 스타일을 추가할 수 있습니다. */
```

---

## 2. 테마 커스터마이징 (선택 사항)

MWDS가 제공하는 기본 오렌지색 대신 프로젝트의 브랜드 컬러를 사용하고 싶다면, CSS 변수를 재정의하면 됩니다.

```css
/* globals.css */
@import "@mwds/theme/src/index.css";

:root {
  /* 브랜드 컬러를 파란색으로 변경 */
  --mw-primary: #0070f3;
  --mw-primary-hover: #0056b3;
  --mw-radius: 12px; /* 둥근 모서리 값 변경 */
}
```

이렇게 하면 모든 MWDS 컴포넌트가 바뀐 색상과 스타일을 자동으로 따르게 됩니다.

---

## 3. 폰트 설정 (권장)

MWDS는 가독성 좋은 디자인을 위해 **Geist** 또는 **Pretendard** 폰트 사용을 권장합니다.

```typescript
// layout.tsx 예시
import { Geist } from "next/font/google";

const geistSans = Geist({
  variable: "--font-geist-sans",
  subsets: ["latin"],
});

export default function RootLayout({ children }) {
  return (
    <html lang="ko">
      <body className={`${geistSans.variable} antialiased font-sans`}>
        {children}
      </body>
    </html>
  );
}
```
