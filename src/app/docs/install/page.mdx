# 설치하기

MWDS(My Design System)를 사용하여 빠르고 일관된 UI를 구축해 보세요.

---

## 1. 개별 패키지 설치

MWDS는 필요한 컴포넌트만 골라서 설치할 수 있는 멀티 패키지 구조로 되어 있습니다.

```bash
# 버튼 컴포넌트 설치
npm install @mwds/button

# 뱃지 컴포넌트 설치
npm install @mwds/badge

# (필수) 공통 테마 설치
npm install @mwds/theme
```

---

## 2. 요구 사항

MWDS는 최신 웹 표준과 라이브러리를 기반으로 합니다.

- **React**: 18.0.0 이상
- **Next.js**: 13.0.0 이상 (App Router 권장)

> **참고**: MWDS는 이제 Tailwind CSS 없이도 독립적으로 동작하도록 설계되었습니다. 하지만 사용자 프로젝트에서 Tailwind를 함께 사용하는 것도 완벽하게 지원합니다.

---

## 3. 피어 종속성 (Peer Dependencies)

MWDS 컴포넌트들은 `react`와 `react-dom`을 피어 종속성으로 가집니다. 일반적인 React/Next.js 프로젝트라면 이미 설치되어 있을 것입니다.

---
